Principales Logros

01

Solución orden de compra e-commerce

Se diseña, propone y desarrolla todo el ciclo de vida de una orden de compra, separando la lógica de negocio e-commerce de procesos ligados a tiendas físicas. Junto con esto se automatizan los procesos de facturación, WMS, tracking, sistemas legados y comunicación a cliente. Logrando así la centralización de información de una orden de compra, flexibilidad para tomar acciones de negocio basándonos en estados, documentos y automatización de procesos. (Project Manager - Tricot S.A.)

02

Solución de tracking

Se diseña y desarrolla sistema de tracking para ordenes de compra e-commerce, integrando proveedores logísticos y negocio, vinculando macroestados de negocio a microestados de couriers. Con esto se logra centralizar los movimientos externos de una orden de compra y la visibilidad para el cliente del estado de su orden. (Project Manager - Tricot S.A.)

03

Versionamiento

Se propone e implementa versionamiento de código fuente en la sub-gerencia de desarrollo y otras áreas de negocio que manejan código fuente. Para esto se realizan talleres y demostraciones de casos de uso, además de acompañamiento y monitorización del cambio. Con esto se solucionan los problemas de incongruencias de código y seguimiento del mismo, además de facilitar los despliegues en los distintos ambientes. (Lead Software Developer - Tricot S.A.)

04

Cambio de lenguaje y paradigma

Se propone e implementa el cambio de lenguaje principal de desarrollo, pasando de PHP 4-5 a Java 8. Para esto se realizan talleres, demostraciones de casos de uso, pruebas de concepto, además de acompañamiento y monitorización del cambio. Con esto se solucionan problemas de seguridad, orden del código, estructuras y se minimizan errores en tiempo de ejecución, entre otras. Junto con esto se cambia el paradigma MVC a uno centrado en crear capas de servicio para las áreas de negocio, y a partir de estas la construcción de aplicativos internos y externos, logrando la extensibilidad e interoperabilidad del negocio tanto dentro de la organización como fuera. (Lead Software Developer - Tricot S.A.)

05

Flujo ágil de desarrollo de software

Se propone, diseña e implementa flujo de desarrollo de software en plataforma Jira. Para realizar este flujo se consideraron las barreras y frenos del flujo actual, dando paso a la agilidad, quitando las barreras burocráticas y dando más autonomía a los equipos. Con esto se agiliza el test y despliegue de los entregables, dando paso a las entregas tempranas que aportan valor al negocio.
